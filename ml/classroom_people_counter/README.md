Проект сейчас представляет собой шаблон системы детекции людей с использованием YOLO. Главный файл main.py запускает программу, принимает параметры из командной строки и, в зависимости от выбранного режима, запускает либо обработку видеофайла, либо работу с видеопотоком камеры. Для режима реального времени используется файл real_time_detector.py, где открывается видеопоток, загружается модель YOLO, считываются кадры и выполняется обнаружение людей. Кадры выводятся на экран, а завершение работы происходит при нажатии клавиши q.
Для обработки заранее записанных роликов предназначен файл video_processor.py. Он также загружает модель, открывает указанный видеофайл и по кадрам выполняет детекцию людей, показывая результат. В коде будет место для сохранения обработанного видео, но эта часть пока не реализована.
Модуль yolo_config.py содержит функцию загрузки YOLO-модели через OpenCV и функцию выполнения детекции.
Утилиты разделены на три модуля. config.py читает YAML-файлы настроек и возвращает их содержимое в виде словаря. visualization.py отвечает за рисование рамок вокруг найденных людей и отображение счётчика на изображении. file_manager.py создает необходимые служебные директории, если их ещё нет.
Конфигурационные файлы в директории config содержат шаблоны настроек для модели, камеры и параметров детекции.
